# 📖 What is LangChain?

LangChain is an open-source framework designed to simplify the development of applications powered by Large Language Models (LLMs).

Instead of writing complex code for every task, LangChain provides reusable components such as Models, Prompts, Chains, Memory, Indexes, and Agents.

It helps developers connect LLMs with external data sources, APIs, tools, databases, and custom workflows.

---

# 🏗 LangChain Architecture

```text
                    LangChain
                         │
 ┌──────────┬─────────┬─────────┬─────────┬─────────┬─────────┐
 │          │         │         │         │         │
Models   Prompts   Chains   Memory   Indexes   Agents
```

Each component performs a specific task, and together they help build intelligent AI applications.

# 1️⃣ Models

Models are responsible for interacting with Large Language Models.

Examples:

- OpenAI GPT
- Claude
- Gemini
- Llama
- Mistral
- Hugging Face Models

LangChain provides a common interface for using different LLM providers without changing much code.

### A. Language Models (LLMs)

Input:

```text

```

Output:

```text

Example:

Input:

Chat Models work with conversations.

Example:

User:

```

# 2️⃣ Prompts

A Prompt is an instruction given to the LLM.

Example:

```
Explain Machine Learning in simple words.
```

LangChain provides PromptTemplate to create reusable prompts.

Example:

```
Explain {topic} in {style}.
```

If:

```
topic = Neural Networks
style = Beginner
```

Final Prompt:

```
Explain Neural Networks in Beginner style.
```

Benefits:

- Reusable
- Dynamic
- Cleaner code
- Easier maintenance
---

# 💻 Prompt Code Examples

## Example 1 – Basic Prompt

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template(
    "Explain {topic} in simple language."
)

formatted_prompt = prompt.format(topic="Machine Learning")

print(formatted_prompt)
```

### Output

```
Explain Machine Learning in simple language.
```

---

## Example 2 – Multiple Variables

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate(
    input_variables=["topic", "level"],
    template="Explain {topic} for a {level} student."
)

formatted_prompt = prompt.format(
    topic="Neural Networks",
    level="Beginner"
)

print(formatted_prompt)
```

### Output

```
Explain Neural Networks for a Beginner student.
```

---

## Example 3 – Role-Based Prompt

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template(
    """
You are an experienced AI professor.

Explain {topic} with an example.
"""
)

print(prompt.format(topic="Embeddings"))
```

### Output

```
You are an experienced AI professor.

Explain Embeddings with an example.
```

---

## Example 4 – Chat Prompt Template

```python
from langchain_core.prompts import ChatPromptTemplate

prompt = ChatPromptTemplate.from_messages(
    [
        ("system", "You are an AI tutor."),
        ("human", "Explain {topic}.")
    ]
)

messages = prompt.format_messages(topic="LangChain")

print(messages)
```

This creates structured chat messages for chat models like GPT, Claude, and Gemini.

# 📌 Why Prompt Templates?

Instead of writing:

```python
prompt = "Explain AI."
```

we can write:

```python
prompt = PromptTemplate.from_template(
    "Explain {topic}."
)
```

Now we can reuse the same prompt for different topics:

```python
prompt.format(topic="AI")
prompt.format(topic="Machine Learning")
prompt.format(topic="Deep Learning")
prompt.format(topic="LangChain")
```

This makes the code:

- Reusable
- Dynamic
- Easier to maintain
- Cleaner

# 3️⃣ Chains

A Chain connects multiple operations together.

Instead of manually calling every function, LangChain executes them sequentially.

Example:

```text
Question

↓

Search PDF

↓

Retrieve Information

↓

Generate Answer

↓

Display Output
```

Chains reduce repetitive coding and make AI workflows easier to manage.

---

# 4️⃣ Memory

LLMs do not remember previous conversations by default.

Memory stores previous interactions so the chatbot can continue conversations naturally.

Types of Memory:

### Conversation Buffer Memory

Stores the complete conversation.

### Conversation Buffer Window Memory

Stores only the most recent interactions.

### Summary Memory

Stores a summarized version of previous conversations to save tokens.

### Custom Memory

Stores custom information such as:

- User Name
- Preferences
- Settings
- Previous Tasks

Memory improves contextual conversations.

---

# 5️⃣ Indexes

Indexes connect external knowledge with the LLM.

Examples of data sources:

- PDF Files
- Word Documents
- CSV Files
- Websites
- Databases

Typical Workflow:

```text
Documents

↓

Document Loader

↓

Text Splitter

↓

Embeddings

↓

Vector Database

↓

Retriever

↓

Relevant Chunks

↓

LLM

↓

Final Answer
```

Indexes are the foundation of Retrieval-Augmented Generation (RAG).

---

# 6️⃣ Agents

Agents are intelligent systems capable of making decisions and using external tools.

Unlike normal chatbots, Agents can decide which tool should be used.

Examples of Tools:

- Calculator
- Weather API
- Google Search
- Database
- File System

Example:

User:

```
What is today's temperature in Delhi?
```

Agent:

```text
Question

↓

Weather API

↓

Get Temperature

↓

Generate Response

↓

Answer User
```

Agents combine reasoning, planning, and tool usage.
